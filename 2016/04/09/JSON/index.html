Javascript Object Notation(扩展名.json)

  • 用于存储文本和交换文本信息，一种格式
  • 比XML更小更快更易解析
    JSON的语法形式与JavaScript对象格式相似(JSON名称必须用””)，JavaScript可以用内建函数eval()来解析JSON，用JSON数据来生成原生的JavaScript对象

eval()方法

缺点：eval()可编译执行任何JS代码，不安全。
更安全的做法：使用JSON解析器。JSON解析器只识别JSON文本，不会编译JS脚本

JSON对象操作

//增
obj.sex = “男”;
或：obj[“sex”] = “男”;
//删
delete obj.sex;
//改
obj.sex = “女”;
或：obj[“sex”] = “女”;
//遍历
for(var c in obj){
console.log(c+”:”+obj[c]);
}
